How it works?

{{ brand }} fetches a set of relevant RSS feeds and ranks the links by the average reach of people using the shares, reactions and comments from Facebook.

Who made it? Donate?

{{ brand }} is product created by Lucian Marin. {{ brand }} is made for busy people who don't want to spent more than a few minutes getting the most important news of the day. You might consider donating for all services provided for free.

When? And why?

Rad-Fi started when I was replying to Elon Musk on Twitter that I can code Pravduh for him. After I exchanged a few DMs with him, I started to get hooked on the idea and after each code iteration it go closer to an actual product.

What's the tech stack?

{{ brand }} is powered by Falcon web framework, Django ORM, feedparser, requests, BeautifulSoup, PostgreSQL, PgBouncer, Facebook API and Google feedburner for feed caching.
